# AI-Emotion-Detection-System
"AI-based Emotion Detection System using DeepFace and OpenCV. Detects human emotions from images in real-time with high accuracy."

🔍 Key Features:
✔️ Real-time Emotion Detection
✔️ Face Analysis using Deep Learning
✔️ Accurate Emotion Classification
✔️ User-friendly Interface

⚙️ Tech Stack:
Python | DeepFace | OpenCV | Computer Vision | Deep Learning
